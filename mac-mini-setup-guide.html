Mac mini M4 Pro — To'liq Setup va Remote Control

🍎 macOS 🔐 SSH 🖥️ Screen Sharing 🌐 Tailscale ⚡ CI/CD
STEP 01

Birinchi marta yoqish

⚠️ Birinchi marta yoqishda monitor va klaviatura ulang. Sozlab bo'lgach ular kerak bo'lmaydi.
  • Mac mini ni yoqib boshlang'ich sozlashni bajaring
  • Apple ID bilan kiring — iCloud va App Store uchun majburiy
  • System Settings → General → Software Update — barcha yangilanishlarni o'rnating
  • Foydalanuvchi nomi va parolni yodda saqlang
  • System Settings → Energy → Wake for network access — ON qiling
  • System Settings → Lock Screen → Require password after sleep — Never qiling
STEP 02

Sharing sozlamalari

System Settings → General → Sharing bo'limini oching:

SozlamaHolatMaqsad
Remote Login✅ ONSSH orqali terminal kirish
Remote Management✅ ONTo'liq grafik boshqaruv
Screen SharingautoRemote Management yoqsa avtomatik
Remote Apple Events✅ ONAppleScript orqali boshqarish
ℹ️ Har birida Allow access for: All users tanlang.
STEP 03

Static IP sozlash

Router ichida (tavsiya)

  • Router admin paneliga kiring: 192.168.2.1
  • DHCP Reservation bo'limini toping
  • Mac mini MAC adresini 192.168.2.126 ga bog'lang

Mac ichida (qo'lda)

System Settings → Network → Wi-Fi → Details → TCP/IP

config
Configure IPv4: Manual
IP Address:     192.168.2.126
Subnet Mask:    255.255.255.0
Router:         192.168.2.1
DNS:            8.8.8.8, 8.8.4.4

Tekshirish

bash
ipconfig getifaddr en1   # WiFi
ipconfig getifaddr en0   # Ethernet
STEP 04

SSH orqali ulanish

bash — MacBook Terminal
# IP orqali
ssh macstation@192.168.2.126

# Hostname orqali
ssh macstation@Fozilbeks-Mac-mini.local
ℹ️ Birinchi ulanishda yes yozing. Keyingi safar so'ramaydi.

Foydali buyruqlar

bash
uptime                          # Tizim holati
top -l 1 | grep PhysMem         # RAM
df -h                           # Disk
ps aux | head -20               # Jarayonlar
sudo shutdown -r now            # Qayta ishga tushirish
sudo shutdown -h now            # O'chirish
STEP 05

Screen Sharing — Ekranni boshqarish

Variant A — Finder orqali

  • MacBook da Finder → Go → Connect to Server (⌘K)
  • vnc://192.168.2.126 kiriting → Connect
  • Username: macstation → parol kiriting

Variant B — Terminal orqali

bash
open vnc://macstation@192.168.2.126

Variant C — Screen Sharing app

  • Spotlight (⌘Space) → Screen Sharing → oching
  • 192.168.2.126 kiriting → Connect
✅ Monitor umuman ulanmagan bo'lsa ham ishlaydi. Mac mini headless rejimda to'liq ishlayveradi.

Monitor yo'q bo'lsa — Virtual Display

bash
# BetterDisplay — virtual monitor yaratadi
brew install --cask betterdisplay

# Yoki $5-10 HDMI dummy plug sotib oling — eng ishonchli yechim
STEP 06

SSH Key — Parolsiz kirish

1. MacBook da key yaratish

bash — MacBook
ssh-keygen -t ed25519 -C "macbook-to-macmini"
# Enter × 3 (passphrase ixtiyoriy)

2. Kalitni Mac mini ga yuborish

bash — MacBook
ssh-copy-id macstation@192.168.2.126

3. SSH Config — alias sozlash

~/.ssh/config
Host macmini
  HostName 192.168.2.126
  User macstation
  IdentityFile ~/.ssh/id_ed25519

Host macmini-remote
  HostName 100.x.x.x
  User macstation
  IdentityFile ~/.ssh/id_ed25519
bash — endi qisqa buyruq
ssh macmini           # Uy tarmog'ida
ssh macmini-remote    # Tashqaridan (Tailscale)

Tekshirish va asosiy paketlar

STEP 08

Tailscale — Tashqaridan kirish

🌐 Tailscale — bepul VPN. Istalgan joydan Mac mini ga ulanish imkonini beradi. Port forwarding shart emas.

Mac mini da o'rnatish

bash — Mac mini (SSH)
brew install tailscale
sudo tailscaled &
sudo tailscale up
# Brauzerda ochilgan link orqali login qiling

MacBook da o'rnatish

bash — MacBook
brew install tailscale
sudo tailscale up
# Bir xil akkauntga kiring

Ulanish

bash
tailscale ip                     # Tailscale IP ni ko'rish
ssh macstation@100.x.x.x         # SSH (istalgan joydan)
open vnc://100.x.x.x             # Ekran (istalgan joydan)
STEP 09

Admin boshqaruv

Tizim boshqaruvi

bash — SSH orqali
sudo softwareupdate -ia                        # Yangilanishlar
open -a "Xcode"                                # Ilova ochish
osascript -e 'quit app "Safari"'               # Ilova yopish
screencapture -x ~/Desktop/screenshot.png      # Screenshot
system_profiler SPHardwareDataType             # Hardware info
sudo lsof -i -P | grep LISTEN                  # Ochiq portlar

Hostname o'zgartirish

bash
sudo scutil --set ComputerName "Fozilbek-Mac-mini"
sudo scutil --set LocalHostName "Fozilbek-Mac-mini"

GitHub Actions Self-hosted Runner

bash
# GitHub → repo → Settings → Actions → Runners → New runner
# macOS ARM64 tanlang, ko'rsatilgan buyruqlarni bajaring

cd ~/actions-runner
./svc.sh install    # Service sifatida o'rnatish
./svc.sh start      # Ishga tushirish
./svc.sh status     # Holat tekshirish
STEP 10

Auto-start va Server sozlamalari

⚠️ Bu buyruqlar server uchun muhim — Mac mini uyquga ketmasin va elektr qaytganda avtomatik yonsin.
bash — bir marta bajarish
sudo pmset -a sleep 0          # Uyquga ketmaslik
sudo pmset -a disksleep 0      # Disk uyquga ketmaslik
sudo pmset -a displaysleep 10  # Ekran 10 daqiqada o'chsin
sudo pmset -a autorestart 1    # Elektr qaytganda avtomatik yoqilish
sudo pmset -a womp 1           # Network wake (WOL)
pmset -g                       # Hozirgi holat

Monitoring

bash
top                            # CPU va RAM real vaqtda
nettop                         # Network trafik
iostat -w 2                    # Disk I/O
tail -f /var/log/system.log    # System log
