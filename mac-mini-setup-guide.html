01
Birinchi marta yoqish
Mac mini ni boshlang'ich sozlash
⚠️ Birinchi marta yoqishda monitor + klaviatura ulang. Keyinchalik ular shart bo'lmaydi.
  • Mac mini ni yoqing va boshlang'ich sozlashni bajaring
  • Apple ID bilan kiring — iCloud, App Store uchun kerak
  • System Settings → General → Software Update — barcha yangilanishlarni o'rnating
  • Foydalanuvchi nomi (macstation) va parolni eslab qoling
  • System Settings → Energy → Wake for network access — ON qiling
  • System Settings → Lock Screen → Require password — "After sleep" → "Never" qiling (server uchun)
02
Sharing sozlamalari
Remote Login, Screen Sharing, Remote Management yoqish

System Settings → General → Sharing bo'limiga kiring:

  • Remote Login → ON — SSH kirish uchun
  • Screen Sharing → ON — ekranni ko'rish uchun
  • Remote Management → ON — to'liq boshqaruv uchun (Screen Sharing o'chib, Remote Management yonadi)
  • Remote Apple Events → ON — AppleScript orqali boshqarish
  • Har birida "Allow access for: All users" tanlang
ℹ️ Remote Management yonsa, Screen Sharing avtomatik o'chadi — bu normal. Remote Management ikkalasini ham qamrab oladi.
03
Static IP sozlash
IP o'zgarmasligi uchun router yoki Mac ichida belgilash

Variant A — Router ichida (tavsiya)

  • Router admin paneliga kiring: 192.168.2.1
  • DHCP Reservation bo'limini toping
  • Mac mini ning MAC adressini toping → 192.168.2.126 ni bog'lang

Variant B — Mac mini ichida

System Settings → Network → Ethernet/Wi-Fi → Details → TCP/IP

Sozlamalar 
Configure IPv4: Manual
IP Address:     192.168.2.126
Subnet Mask:    255.255.255.0
Router:         192.168.2.1
DNS:            8.8.8.8, 8.8.4.4

Tekshirish

Terminal — Mac mini 
# WiFi orqali
ipconfig getifaddr en1

# Ethernet orqali
ipconfig getifaddr en0

05
Screen Sharing — Ekranni ko'rish
MacBook dan Mac mini ekranini to'liq boshqarish

Variant A — macOS Screen Sharing (eng oson)

  • MacBook da Finder → Go → Connect to Server (⌘K)
  • Manzil kiriting: vnc://192.168.2.126
  • Username: macstation — Parol: Mac mini paroli

Variant B — Terminal orqali

MacBook — Terminal 
open vnc://macstation@192.168.2.126

Variant C — Screen Sharing ilovasi

  • MacBook da Spotlight (⌘Space) → "Screen Sharing" yozing → oching
  • 192.168.2.126 kiriting → Connect
✅ Monitor umuman ulanmagan bo'lsa ham ishlaydi! Mac mini headless (monitorsiz) rejimda ishlayveradi.

Headless — Virtual monitor (monitor yo'q bo'lsa)

Agar Mac mini ga monitor ulanmagan bo'lsa, ekran o'lchami kichik bo'lishi mumkin. Buni hal qilish uchun:

SSH orqali Mac mini da 
# BetterDisplay yoki DisplayBuddy orqali virtual display yaratish
# Yoki System Settings → Displays → Resolution ni o'zgartiring
sudo /usr/bin/defaults write /Library/Preferences/com.apple.windowserver \
  DisplayResolutionEnabled -bool true
06
Homebrew o'rnatish
Paket menejeri — barcha toollar shu orqali o'rnatiladi
⚠️ O'rnatish 10–15 daqiqa ketadi. SSH sessiyasini yopmang!

1. O'rnatish

Mac mini — Terminal yoki SSH 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. PATH qo'shish (M4 Pro — ARM chip uchun)

O'rnatib bo'lgach 
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"

3. Tekshirish

Versiyani ko'rish 
brew --version
# Homebrew 4.x.x chiqishi kerak
07
Tailscale — Tashqaridan kirish
Uydan tashqarida (ofis, ko'cha) ham Mac mini ga ulanish
🌐 Tailscale — bepul VPN. Mac mini va MacBook ni bitta virtual tarmoqqa bog'laydi. Port forwarding shart emas.

1. Mac mini da o'rnatish (SSH orqali)

Mac mini 
brew install tailscale

# Tailscale ni ishga tushirish
sudo tailscaled &

# Login qilish (link ochiladi)
sudo tailscale up

2. MacBook da o'rnatish

MacBook 
# App Store dan ham o'rnatish mumkin
brew install tailscale
sudo tailscale up

3. Tailscale IP ni ko'rish

Mac mini da 
tailscale ip
# 100.x.x.x formatida IP chiqadi

4. Istalgan joydan ulanish

MacBook — istalgan tarmoqdan 
# SSH orqali
ssh macstation@100.x.x.x

# Yoki hostname orqali
ssh macstation@fozilbeks-mac-mini

# Screen sharing
open vnc://100.x.x.x
✅ Tailscale bepul, 3 ta qurilmaga qadar. Ikkalangizda ham bir xil Tailscale akkauntga kiring.

Masofadan Admin boshqaruv
sudo, tizim sozlamalari, ilovalar o'rnatish — to'liq huquq

SSH orqali sudo ishlatish

Admin buyruqlar 
# Tizim yangilanishlarini tekshirish
sudo softwareupdate -l

# Yangilanishlarni o'rnatish
sudo softwareupdate -ia

# Ilova o'rnatish (brew orqali)
brew install --cask visual-studio-code
brew install --cask docker
brew install git node python

# Xizmatlarni boshqarish
sudo launchctl list
sudo launchctl start com.example.service

# Firewall holati
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --getglobalstate

# Tizim ma'lumotlari
system_profiler SPHardwareDataType

Masofadan ilova ochish va yopish

App boshqaruv — SSH orqali 
# Ilova ochish
open -a "Safari"
open -a "Xcode"

# Ilova yopish
osascript -e 'quit app "Safari"'

# Barcha ochiq ilovalar
osascript -e 'tell application "System Events" to get name of every process where background only is false'

# Screenshot olish (masofadan)
screencapture -x ~/Desktop/screenshot.png

Tizim sozlamalarini masofadan o'zgartirish

System preferences — Terminal orqali 
# Hostname o'zgartirish
sudo scutil --set ComputerName "Fozilbek-Mac-mini"
sudo scutil --set LocalHostName "Fozilbek-Mac-mini"

# Avtomatik uyqu o'chirish (server uchun muhim!)
sudo pmset -a sleep 0
sudo pmset -a disksleep 0
sudo pmset -a displaysleep 0

# Network qayta ishga tushirish
sudo ifconfig en0 down && sudo ifconfig en0 up
⚠️ Server uchun muhim: sudo pmset -a sleep 0 buyrug'ini albatta ishlatib qo'ying — Mac mini uyquga ketmasin.
09
SSH Key — Parolsiz kirish
Har safar parol yozmaslik uchun SSH key sozlash

1. MacBook da SSH key yaratish

MacBook — Terminal 
# Key yaratish (ed25519 — eng xavfsiz)
ssh-keygen -t ed25519 -C "macbook-to-macmini"

# Enter bosib davom eting (passphrase ixtiyoriy)

2. Kalitni Mac mini ga yuborish

MacBook — Terminal 
ssh-copy-id macstation@192.168.2.126

# Yoki qo'lda:
cat ~/.ssh/id_ed25519.pub | ssh macstation@192.168.2.126 \
  "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys"

3. Endi parolsiz kirish

MacBook — parol so'ralmaydi 
ssh macstation@192.168.2.126
# To'g'ridan kiradi!

4. SSH Config (alias) sozlash

~/.ssh/config — MacBook da 
Host macmini
  HostName 192.168.2.126
  User macstation
  IdentityFile ~/.ssh/id_ed25519

# Tailscale orqali ham
Host macmini-remote
  HostName 100.x.x.x
  User macstation
  IdentityFile ~/.ssh/id_ed25519
Endi qisqa buyruq bilan kirish 
# Uy tarmog'ida
ssh macmini

# Tashqaridan (Tailscale)
ssh macmini-remote
10
Auto-start va Server sozlamalari
Mac mini o'chib yonsa ham hamma narsa avtomatik ishga tushsin

Elektr o'chib yonsa avtomatik yoqilishi

  • System Settings → General → Startup Disk
  • System Settings → Energy → "Start up automatically after a power failure" → ON

Terminal orqali

Avtomatik yoqilish sozlash 
# Elektr qaytganda avtomatik yoqilish
sudo pmset -a autorestart 1

# Uyquga ketmaslik
sudo pmset -a sleep 0
sudo pmset -a disksleep 0
sudo pmset -a displaysleep 10

# Network wake (WOL)
sudo pmset -a womp 1

# Hozirgi holat
pmset -g

Docker auto-start (CI/CD uchun)

Docker Desktop sozlamasi 
# Docker o'rnatish
brew install --cask docker

# Docker Desktop → Settings → General:
# ✅ Start Docker Desktop when you sign in to your computer

GitHub Actions Self-hosted Runner

CI/CD Runner o'rnatish 
# GitHub repo → Settings → Actions → Runners → New self-hosted runner
# macOS ARM64 tanlang, ko'rsatilgan buyruqlarni bajaring

# Runner ni service sifatida o'rnatish (auto-start)
cd ~/actions-runner
./svc.sh install
./svc.sh start

# Holat tekshirish
./svc.sh status

Foydali monitoring buyruqlari

Tizim monitoring 
# CPU, RAM real vaqtda
top

# Faqat CPU va RAM summary
vm_stat && top -l 1 | grep -E "^CPU|^PhysMem"

# Network trafik
nettop

# Disk I/O
iostat -w 2

# Log fayllarni ko'rish
tail -f /var/log/system.log

# Ochiq portlar
sudo lsof -i -P | grep LISTEN
